# Dependencies

- Node.js
- SQLite3

# Build/Run Instructions

Install the app's dependencies from within the Photify directory:
```
npm install

```

Run the app:
```
npm start
```
Now you should be able to navigate to http://localhost:3000/ on your favorite browser and see the app running.
